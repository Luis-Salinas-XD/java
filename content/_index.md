### Variables y Constantes

- Las variables son espacios de memoria reservados para un valor, este puede ser reasignalble, se pueden declarar con **_"var"_** ó **_"let"_**.

```
let nombre = "Luis";
var apellido = prompt('¿Cual es su apellido?');
```
